A simple limiting regime for studying generalization analytically

Wide neural network in the NTK regime $\approx$ Kernel regression

To better understand generalization, one useful approach is to consider a limiting setting in which a wide neural network becomes equivalent to kernel regression, a classical and analytically tractable statistical model.

[Setup] Consider a fully connected $L$-layer multilayer perceptron (MLP) with input dimension $d$, elementwise nonlinearity $\sigma$, hidden-layer widths $n_1,\ldots,n_{L-1}$, and a scalar output. For an input $\mathbf{x}\in\mathbb{R}^{d}$, the activation $h_\ell(\mathbf{x})$ at layer $\ell$ and the network output $f(\mathbf{x})$ are defined recursively as:

$$h_0(\mathbf{x})=\mathbf{x},\qquad h_{\ell+1}(\mathbf{x})=\sigma\!\left(\frac{1}{\sqrt{n_\ell}}W^{(\ell)}h_\ell(\mathbf{x})\right) \quad (\ell=0,\ldots,L-1),\qquad f(\mathbf{x})=\frac{1}{\sqrt{n_L}}\,w^\top h_L(\mathbf{x}).$$

We train the MLP with dataset $\mathscr{D}=\{(\mathbf{x}_i,y_i)\}_{i=1}^N$ sampled independently from the data distribution, with $y_i=f_T(\mathbf{x}_i)$. Denote $\mathbf{X}=\{\mathbf{x}_1,\ldots,\mathbf{x}_N\}$ as the training inputs, $\mathbf{y}=(y_1,\ldots,y_N)^\top$ as the corresponding target vector, and $\theta=\operatorname{vec}\{W^{(0)},\ldots,W^{(L-1)},w\}\in\mathbb{R}^{P\times 1}$ as the vectorization of all trainable weights. The training loss $\mathcal{L}(\theta)$ is defined as the empirical squared loss:

$$\mathcal{L}(\theta)=\frac{1}{2}\left\lVert f_\theta(\mathbf{X})-\mathbf{y}\right\rVert_2^2.$$

In continuous time, the parameters update according to gradient descent with learning rate $\eta$:

$$\dot{\theta}_t=-\eta\nabla_\theta\mathcal{L}(\theta_t).$$

[Training Dynamics] Our goal is to find the analytical expression for the learned function $f$ after training. To track how the network output function $f$ evolves during training, we first evaluate it on the training inputs $\mathbf{X}$. Apply the chain rule to transform gradient flow in parameter space into dynamics in function space:

$$\dot{f}_t(\mathbf{X}) \overset{\color{gray}{\text{chain rule}}}{=} \dimensioncolor{\underbrace{\!\vphantom{\nabla_\theta f_t(\mathbf{X})\nabla_\theta f_t(\mathbf{X})^\top}\color{black}{\nabla_\theta f_t(\mathbf{X})}\!}_{\mathbb{R}^{N\times P}}}\; \dimensioncolor{\underbrace{\vphantom{\nabla_\theta f_t(\mathbf{X})\nabla_\theta f_t(\mathbf{X})^\top}\color{black}{\dot{\theta}_t}}_{\substack{\\[-9pt]\mathbb{R}^{P\times 1}}}} \overset{\color{gray}{\text{gradient flow}}}{=}-\eta\, \dimensioncolor{\underbrace{\vphantom{\nabla_\theta f_t(\mathbf{X})\nabla_\theta f_t(\mathbf{X})^\top}\rule[-3pt]{0pt}{0pt}\mkern-3mu\color{black}{\nabla_\theta f_t(\mathbf{X})\nabla_\theta f_t(\mathbf{X})^\top}\mkern-3mu}_{\text{NTK}\in\mathbb{R}^{N\times N}}} \hspace{0.35em}\dimensioncolor{\underbrace{\vphantom{\nabla_\theta f_t(\mathbf{X})\nabla_\theta f_t(\mathbf{X})^\top}\mkern-3mu\color{black}{\nabla_{f_t(\mathbf{X})}\mathcal{L}}\mkern-3mu}_{\substack{\\[-2pt]\mathbb{R}^{N\times 1}}}} \overset{\color{gray}{\text{define NTK}}}{=}-\eta\,K_t(\mathbf{X},\mathbf{X})\nabla_{f_t(\mathbf{X})}\mathcal{L}.$$

[NTK] The key object that appears in the expansion above is the Gram matrix of the Jacobian, which can be interpreted as a kernel function evaluated on pairs of training data. A kernel is a function that takes two inputs and returns a scalar similarity measure. It can be written as an inner product between feature vectors $\phi$: $K(\mathbf{x},\mathbf{x}')=\langle \phi(\mathbf{x}), \phi(\mathbf{x}') \rangle$. In our case, the feature map is the Jacobian $\phi(\mathbf{x})=\nabla_\theta f_t(\mathbf{x})$, and the resulting kernel was first defined as the Neural Tangent Kernel (NTK) by Jacot et al. (2018):

$$K_t(\mathbf{x},\mathbf{x}') := \dimensioncolor{\underbrace{\vphantom{\nabla_\theta f_t(\mathbf{x})^\top}\color{black}{\nabla_\theta f_t(\mathbf{x})^\top}}_{\mathbb{R}^{1\times P}}} \hspace{0.25em}\dimensioncolor{\underbrace{\vphantom{\nabla_\theta f_t(\mathbf{x})^\top}\color{black}{\nabla_\theta f_t(\mathbf{x}')}}_{\mathbb{R}^{P\times1}}} =\sum_{p=1}^{P} \frac{\partial f_t(\mathbf{x})}{\partial\theta_p} \frac{\partial f_t(\mathbf{x}')}{\partial\theta_p}.$$

If we let all hidden-layer widths $n_\ell\to\infty$, initialize the weights as $W_{ij}^{(\ell)}\sim\mathcal{N}(0,1)$, and use a small learning rate, the NTK converges to a deterministic kernel at initialization and remains constant throughout training ( Jacot et al., 2018 ; Lee et al., 2019 ). This setting, in which the NTK remains stationary during training, is often referred to as the NTK regime, or lazy regime. The $1/\sqrt{n_\ell}$ scaling in the network preactivations is key to lazy learning: under this parameterization, the kernel's deviation from its initialization is bounded as $\sup_{t\geq 0}\lVert K_t-K_0\rVert_F=\mathcal{O}\!\left(1/\sqrt{n}\right)$. This deviation vanishes as $n\to\infty$, so the NTK remains fixed at its initial value throughout training:

$$K_t\overset{n\to\infty}{\longrightarrow} K_0.$$

[Solving the Dynamics with Stationary NTK] With the fixed NTK, the function-space dynamics therefore reduce to linear ordinary differential equations that can be solved analytically. Denote $K=K_0(\mathbf{X},\mathbf{X})$ as the stationary NTK Gram matrix on the training inputs. Since the weights are initialized from a mean-zero Gaussian distribution, the network function $f$ at initialization converges to a mean-zero Gaussian process. For simplicity, we study the centered dynamics and take $f_0=0$.

The training dynamics are solved as:

$$\dot f_t(\mathbf{X}) = -\eta K\nabla_{f_t(\mathbf{X})}\mathcal{L} \quad\overset{\color{gray}{\nabla_{f_t(\mathbf{X})}\mathcal{L}=f_t(\mathbf{X})-\mathbf{y}}}{\Longrightarrow}\quad \dot f_t(\mathbf{X}) = -\eta K\bigl(f_t(\mathbf{X})-\mathbf{y}\bigr) \quad\Longrightarrow\quad f_t(\mathbf{X}) = \left(I-e^{-\eta Kt}\right)\mathbf{y}.$$

The dynamics for a test input $\mathbf{x}_{\mathrm{test}}$ are solved as:

$$\dot f_t(\mathbf{x}_{\mathrm{test}}) = -\eta k_{\mathrm{test}}^\top\bigl(f_t(\mathbf{X})-\mathbf{y}\bigr) \quad\Longrightarrow\quad f_t(\mathbf{x}_{\mathrm{test}}) = \eta k_{\mathrm{test}}^\top\int_0^t e^{-\eta Ks}\,ds\,\mathbf{y} \quad\overset{\color{gray}{\eta\int_0^t e^{-\eta Ks}\,ds=K^{-1}\left(I-e^{-\eta Kt}\right)}}{\Longrightarrow}\quad f_t(\mathbf{x}_{\mathrm{test}}) = k_{\mathrm{test}}^\top K^{-1} \left(I-e^{-\eta Kt}\right)\mathbf{y},$$

where the test-kernel vector $k_{\mathrm{test}}=K_0(\mathbf{X},\mathbf{x}_{\mathrm{test}})=\bigl[K_0(\mathbf{x}_1,\mathbf{x}_{\mathrm{test}}),\ldots,K_0(\mathbf{x}_N,\mathbf{x}_{\mathrm{test}})\bigr]^\top\in\mathbb{R}^{N\times1}$.

[Learned Predictor = Kernel Regression Solution] If $K$ is positive definite, then $e^{-\eta Kt}\to \mathbf{0}$ as $t\to\infty$. The converged predictor therefore satisfies

$$f_\infty(\mathbf{X})=\mathbf{y},\qquad f_\infty(\mathbf{x}_{\mathrm{test}}) =k_{\mathrm{test}}^\top K^{-1}\mathbf{y}.$$

The prediction at the test point exactly matches the ridgeless kernel regression solution, or equivalently the zero-regularization limit of kernel ridge regression (KRR), a well-studied nonparametric statistical method in function space. This equivalence allows us to approximate the behavior of a wide MLP using KRR and analyze its generalization through the established theory of kernel methods.
